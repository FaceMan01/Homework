Для создания новых пользователей в Linux используются команды useradd или adduser.

Для создания или изменения пароля для указанного пользователя используется команда passwd.

Для выполнения действий от имени другого пользователя без завершения/смены сеанса пользователя используется команда sudo.

Путь к домашней директории пользователя можно определить, воспользовавшись переменной окружения $HOME, либо командой echo ~username, где username - имя пользователя.

Команда exit выполняет выход из окружения пользователя и завершает текущий сеанс.

Для узнавания имени текущего пользователя используется команда whoami.

Для смены владельца файла используется команда chown.

Для изменения прав доступа к файлам используется команда chmod.

Для скачивания файлов по ссылкам можно использовать утилиту wget или curl.

Менеджер управления пакетами, установленный по умолчанию в дистрибутиве Ubuntu, называется apt (или apt-get), который используется для установки, удаления и обновления программного обеспечения.
