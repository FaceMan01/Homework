Команда `echo` используется для вывода текста в командную строку.

Для вывода содержимого файла в командную строку можно использовать команду `cat` или `less`.

Команда whoami используется для определения имени учетной записи пользователя.

Для переименования файлов или каталогов используется команда `mv` (от "move").

Для копирования файлов или каталогов при помощи командной строки используется команда `cp` (от "copy").

Для создания новых файлов в командной строке используется команда `touch`, а для создания новых каталогов - команда `mkdir`.

Для удаления файлов используется команда `rm`, а для удаления каталогов - команда `rmdir` (если каталог пустой) или `rm -r` (для удаления каталога и его содержимого).

Команда `ls` позволяет просмотреть содержимое указанного ей каталога (пути).

Для вывода содержимого каталогов, включая подкаталоги, в древовидном формате используется команда `tree`. Если она не установлена, то можно использовать `find` с параметрами, например: `find . -type d`.
