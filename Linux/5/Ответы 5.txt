Сетевой интерфейс - это аппаратное или программное устройство, которое позволяет компьютеру подключаться к сети и обмениваться данными с другими устройствами в этой сети.

При применении утилиты ifconfig без каких-либо параметров будет выведена информация о всех сетевых интерфейсах на компьютере, включая их IP-адреса, MAC-адреса и статус соединения.

Утилита ip link show покажет информацию о всех сетевых интерфейсах, включая их состояние и принадлежность к физическим устройствам.

При применении утилиты ip -br a sh будет выведен краткий список сетевых интерфейсов с их параметрами в упрощенном, "компактном" формате.

Объект: ip, команда: addr add, параметры команды: 192.168.1.11/255.255.255.240 (IP-адрес и маска подсети), dev enp0s8 (название сетевого интерфейса).

Формат вызова команды для удаления IP-адреса: sudo ip addr del <IP-адрес>/<маска> dev <название интерфейса>.

Формат вызова команды для включения/отключения IP-адреса: sudo ip link set <название интерфейса> up (включение) и sudo ip link set <название интерфейса> down (отключение).

Для просмотра маршрутов в таблице маршрутизации можно использовать команду ip route или netstat -r.

Формат вызова команды для добавления маршрута: sudo ip route add <целевая_сеть> via <шлюз> dev <интерфейс>; для удаления маршрута: sudo ip route del <целевая_сеть>.

Команды netstat и ss используются для отображения различной сетевой статистики, такой как открытые сетевые соединения, маршруты и многое другое.

Основная разница между netstat и ss заключается в том, что ss является более современной и быстрой альтернативой netstat. ss предоставляет более подробную информацию о сетевых соединениях и более эффективен в работе с большим количеством соединений.
